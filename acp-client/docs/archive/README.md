# Архив документации клиента ACP

**Дата архивирования:** 13 апреля 2026

## Описание

Эта директория содержит архивированные документы клиента ACP Protocol, сохраненные для исторической справки. Документы отражают предыдущие версии спецификации, дизайн и подходы, которые были замещены на актуальную версию.

## Содержание

### MIGRATION-FROM-OLD-API.md
Руководство по миграции с предыдущей версии API клиента на новую версию, включая инструкции по обновлению кода.

### TUI_CLIENT_SPECIFICATION.md
Спецификация текстового пользовательского интерфейса (TUI) клиента, описание компонентов и взаимодействия.

## Использование

Архивированные документы предназначены для:
- **Исторической справки** — понимание эволюции API и интерфейса клиента
- **Контекста разработки** — анализ причин смены подходов и дизайна
- **Обучения** — изучение истории развития клиента и интеграции с протоколом
