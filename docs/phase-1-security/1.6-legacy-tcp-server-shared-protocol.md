# 1.6 — Shared `ACPProtocol` в legacy TCP-сервере

**Приоритет:** 🟡 Средний  
**Оценка:** 2 часа  
**Файл:** `src/codelab/server/server.py`

---

## Проблема

Legacy TCP-сервер создаёт один экземпляр `ACPProtocol` для всех подключённых клиентов:

```python
class ACPServer:
    def __init__(self, host: str = "127.0.0.1", port: int = 8765) -> None:
        self.protocol = ACPProtocol()  # ← один на всех!

    async def handle_client(self, reader, writer):
        while True:
            raw = await reader.readline()
            outcome = await self.protocol.handle(request)  # ← race condition
```

`ACPProtocol` содержит изменяемое состояние:
- `_sessions: dict[str, SessionState]` — общий для всех клиентов
- `_authenticated: bool` — один флаг на всех клиентов
- `_runtime_capabilities` — перезаписывается каждым новым клиентом

При двух одновременных соединениях:
- Клиент A авторизуется — `_authenticated = True`
- Клиент B без авторизации получает доступ к сессиям клиента A
- Клиент A устанавливает `_runtime_capabilities` — клиент B получает чужие capabilities

---

## Решение

Каждое TCP-соединение должно получать свой экземпляр `ACPProtocol`.

```python
# БЫЛО:
class ACPServer:
    def __init__(
        self,
        host: str = "127.0.0.1",
        port: int = 8765,
    ) -> None:
        self.host = host
        self.port = port
        self.protocol = ACPProtocol()  # ← удалить

    async def handle_client(self, reader, writer):
        peer = writer.get_extra_info("peername")
        try:
            while True:
                ...
                outcome = await self.protocol.handle(request)
```

```python
# СТАЛО:
from ..storage import SessionStorage, InMemoryStorage

class ACPServer:
    def __init__(
        self,
        host: str = "127.0.0.1",
        port: int = 8765,
        *,
        storage: SessionStorage | None = None,
        require_auth: bool = False,
        auth_api_key: str | None = None,
    ) -> None:
        self.host = host
        self.port = port
        # Хранилище сессий — shared между соединениями (персистентность)
        self._storage = storage or InMemoryStorage()
        self._require_auth = require_auth
        self._auth_api_key = auth_api_key
        # ACPProtocol создаётся per-connection в handle_client()

    async def handle_client(
        self, reader: asyncio.StreamReader, writer: asyncio.StreamWriter
    ) -> None:
        """Каждое соединение получает изолированный экземпляр протокола."""
        peer = writer.get_extra_info("peername")

        # Изолированный экземпляр для этого соединения
        protocol = ACPProtocol(
            require_auth=self._require_auth,
            auth_api_key=self._auth_api_key,
            storage=self._storage,   # хранилище shared — сессии персистентны
        )

        try:
            while True:
                raw = await reader.readline()
                if not raw:
                    break
                method_name: str | None = None

                try:
                    request = ACPMessage.from_json(raw.decode().strip())
                    method_name = request.method
                    outcome = await protocol.handle(request)
                except (json.JSONDecodeError, KeyError, TypeError, ValidationError) as exc:
                    outcome = ProtocolOutcome(
                        response=ACPMessage.error_response(
                            None, code=-32700, message="Parse error", data=str(exc),
                        )
                    )

                for notification in outcome.notifications:
                    writer.write((notification.to_json() + "\n").encode())
                    await writer.drain()

                if outcome.response is not None:
                    writer.write((outcome.response.to_json() + "\n").encode())
                    await writer.drain()

                if method_name == "shutdown":
                    break
        finally:
            # Отменить активные turns при разрыве соединения
            await protocol.cancel_active_turns_on_disconnect()
            writer.close()
            await writer.wait_closed()
```
