# Архив документации проекта ACP Protocol

**Дата архивирования:** 13 апреля 2026

## Описание

Эта директория содержит архивированные документы, которые сохранены для исторической справки и справочного материала. Документы в архиве отражают промежуточные этапы разработки, анализа и планирования проекта ACP Protocol.

## Структура архива

### [analysis/](./analysis/)
Аналитические документы и исследования по протоколу ACP:
- Анализ спецификации протокола
- Исследование совместимости с RooCode
- Архитектура наивного агента
- Отчеты о соответствии и комплиансе

### [refactoring/](./refactoring/)
Документы планирования рефакторинга архитектуры обработчиков и реализации агента:
- Планы переструктурирования обработчиков
- Стратегии реализации компонентов

### [integration/](./integration/)
Руководства по интеграции и спецификации для взаимодействия компонентов:
- Интеграция транспорта агента
- Планы интеграции LLM провайдеров
- Спецификации взаимодействия

### [diagrams/](./diagrams/)
Диаграммы взаимодействия и архитектуры системы:
- Диаграммы взаимодействия компонентов

## Использование

Документы в этом архиве предназначены для:
- **Исторической справки** — понимание эволюции проекта

- **Обучения** — изучение подходов и решений при разработке
